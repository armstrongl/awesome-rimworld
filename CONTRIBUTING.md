# Contribution Guidelines

Thank you for contributing to this curated list of RimWorld mods, tools, and resources! Please follow these guidelines to help maintain quality and consistency.

## Adding an Item

### Requirements

- The item must be relevant to RimWorld (mods, tools, guides, or community resources)
- You should have personal experience with the item
- **Quality criteria for mods:**
  - Actively maintained (updated within the last year preferred)
  - Good community reputation (positive reviews, active discussion)
  - Compatible with the current RimWorld version (1.5+)
  - No known game-breaking bugs or conflicts with popular mods

### Format

All items must follow this format:

```markdown
- [Item Name](https://url.com) - Brief description ending with a period.
```

**Important formatting rules:**

- Use `[Name](URL) - Description` format (space-dash-space between URL and description)
- Descriptions must end with proper punctuation (period, exclamation, or question mark)
- Keep descriptions concise (one sentence)
- Place items in alphabetical order within their category

### Process

1. Search existing items to avoid duplicates
2. Check existing issues and PRs to avoid duplicate work
3. Create a pull request with your addition
4. Ensure the PR checks pass (linting, formatting)

## Suggesting Changes

- For new categories: Open an issue first to discuss
- For restructuring: Open an issue explaining your rationale
- For removing items: Open an issue with evidence (broken links, abandoned project, etc.)

## Reporting Issues

- **Dead links:** Use the dead link issue template
- **Other issues:** Open a general issue with details

## Code of Conduct

Be respectful and constructive. This list is meant to help everyone discover great resources.
