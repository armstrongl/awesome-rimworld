## Official Resources

Official RimWorld resources from Ludeon Studios.

- [Ludeon Studios](https://ludeon.com/) - The developer of RimWorld.
- [RimWorld Steam Store](https://store.steampowered.com/app/294100/RimWorld/) - Official Steam page for purchasing RimWorld.
- [RimWorld Wiki](https://rimworldwiki.com/) - Community-maintained wiki with comprehensive game documentation.

## Mod Managers

Tools for managing, sorting, and organizing your mod lists.

- [RimPy Mod Manager Database](https://steamcommunity.com/sharedfiles/filedetails/?id=1847679158) - Community-maintained database of mod sorting rules used by both RimPy and RimSort.
- [RimSort](https://github.com/RimSort/RimSort) - Open-source mod manager with cross-platform support, actively maintained as the modern alternative to RimPy.

## Framework Mods

Core libraries and frameworks required by many mods.

- [Harmony](https://github.com/pardeike/Harmony) - The essential patching library that most RimWorld mods depend on.
- [HugsLib](https://steamcommunity.com/sharedfiles/filedetails/?id=818773962) - Shared library providing common functionality for many mods.
- [Vanilla Expanded Framework](https://steamcommunity.com/sharedfiles/filedetails/?id=2023507013) - Required framework for all Vanilla Expanded mods with shared features like custom factions and building exports.

## Quality of Life

Mods that improve the day-to-day experience of playing RimWorld.

- [Allow Tool](https://steamcommunity.com/sharedfiles/filedetails/?id=761421485) - Adds mass allow/forbid tools, priority hauling, and batch commands for hunting, harvesting, and more.
- [CM Color Coded Mood Bar](https://steamcommunity.com/sharedfiles/filedetails/?id=2006605356) - Color-codes colonist mood bars from green (happy) to red (mental break imminent).
- [Common Sense](https://steamcommunity.com/sharedfiles/filedetails/?id=1561769193) - Makes colonists smarter with optional behaviors like cleaning before work.
- [Deep Storage](https://steamcommunity.com/sharedfiles/filedetails/?id=1617282896) - Adds storage buildings like shelves and pallets to dramatically reduce storage room sizes.
- [Dubs Mint Menus](https://steamcommunity.com/sharedfiles/filedetails/?id=1446523594) - Overhauls architect and crafting menus with searching and filtering.
- [EdB Prepare Carefully](https://steamcommunity.com/sharedfiles/filedetails/?id=735106432) - Fine-tuned control over starting colonist creation with balanced and non-balanced options.
- [Pick Up And Haul](https://steamcommunity.com/sharedfiles/filedetails/?id=1279012058) - Colonists fill their inventory when hauling instead of carrying one item at a time.
- [Replace Stuff](https://steamcommunity.com/sharedfiles/filedetails/?id=1372003680) - Upgrade existing walls and furniture in-place without deconstructing first.
- [Smart Medicine](https://steamcommunity.com/sharedfiles/filedetails/?id=1309994319) - Doctors use medicine from patient inventory during surgery.
- [Stack XXL](https://steamcommunity.com/sharedfiles/filedetails/?id=826366050) - Configurable stack size multipliers to reduce storage space requirements.
- [Wall Light](https://steamcommunity.com/sharedfiles/filedetails/?id=1423699208) - Wall-mounted lights that don't take up floor space.

## Performance

Mods that improve game performance, especially for large colonies and mod lists.

- [Performance Fish](https://github.com/bbradson/Performance-Fish) - Improves framerates and tick times by patching methods for efficiency, compatible with most mods including CE.
- [Performance Optimizer](https://steamcommunity.com/sharedfiles/filedetails/?id=2664723367) - Various performance tweaks providing around 20-35% performance gain.
- [RocketMan](https://steamcommunity.com/sharedfiles/filedetails/?id=2479389928) - Major performance improvements including cached def loading for faster startup with large mod lists.

## UI and Interface

Mods that enhance the user interface and information display.

- [Designator Shapes](https://steamcommunity.com/sharedfiles/filedetails/?id=1235618154) - Adds circle, oval, and other shapes to the building designator.
- [Numbers](https://steamcommunity.com/sharedfiles/filedetails/?id=1864442972) - Adds spreadsheet-like colonist overview with sortable columns for stats.
- [RimHUD](https://steamcommunity.com/sharedfiles/filedetails/?id=1508850027) - Sleek UI overlay showing colonist stats, moods, needs, and injuries at a glance.

## Combat

Mods that overhaul or enhance RimWorld's combat system.

- [Avoid Friendly Fire](https://steamcommunity.com/sharedfiles/filedetails/?id=1134165362) - Colonists avoid shooting when allies are in the line of fire.
- [Combat Extended](https://steamcommunity.com/sharedfiles/filedetails/?id=1631756268) - Comprehensive combat overhaul with ammunition, armor penetration, and realistic ballistics.
- [Run and Gun](https://steamcommunity.com/sharedfiles/filedetails/?id=1204108550) - Allows pawns to fire ranged weapons while moving.
- [Search and Destroy](https://steamcommunity.com/sharedfiles/filedetails/?id=1467764609) - Toggle mode for drafted pawns to automatically hunt down enemies.
- [Yayo's Combat](https://steamcommunity.com/sharedfiles/filedetails/?id=2038409475) - Lighter combat overhaul with new animations and optional ammo system.

- [Vanilla Furniture Expanded](https://steamcommunity.com/sharedfiles/filedetails/?id=1718190143) - Adds diverse furniture options for building varied colonies.

## Content Expansion

Major mods that add significant new content or mechanics.
